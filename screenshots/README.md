# Screenshots

Bu klasöre projenin ekran görüntülerini ekleyebilirsin.

## Önerilen Ekran Görüntüleri

1. **Ana Sayfa** (`homepage.png` veya `homepage.jpg`)
   - Belge seçim ekranı
   - Zorunlu belgeler listesi

2. **Yükleme Sayfası** (`upload.png`)
   - Belge yükleme arayüzü
   - Onay kutuları

3. **Sonuç Sayfası** (`result.png`)
   - Hazırlık skoru
   - Belge detayları
   - Risk değerlendirmesi

4. **Belge Tanıma Örneği** (`document-recognition.png`)
   - Pasaport tanıma örneği
   - Banka dökümü analizi

## Resim Formatı

- Format: PNG veya JPG
- Boyut: Maksimum 1920x1080 (Full HD)
- Dosya boyutu: Maksimum 500 KB (optimize edilmiş)
- İsimlendirme: Küçük harf, tire ile ayrılmış (örn: `homepage-screenshot.png`)

## Resim Optimizasyonu

Resimleri optimize etmek için:

```bash
# ImageMagick ile (eğer yüklüyse)
convert screenshot.png -resize 1920x1080 -quality 85 optimized.png

# veya online araçlar kullan
# - TinyPNG
# - Squoosh
# - ImageOptim
```
